# hensachi-lab 計画書

世の中の正規分布に基づくデータを使い、入力した値の **偏差値・パーセンタイル・z スコア** などを
ブラウザ内の DuckDB (DuckDB-WASM) で計算する統計サイト。

## 決定事項

| 項目 | 選択 | 理由 |
| --- | --- | --- |
| データテーマ | 色々混ぜる汎用スキーマ | 身長/体重/IQ/握力/50m走 など後から足せる形にする |
| アーキテクチャ | ブラウザ完結 (DuckDB-WASM) | サーバ不要・静的ホスティングで公開できる。「ローカルで DB 管理」に合致 |
| 元データ | 公表平均/SD から疑似サンプル生成 | DuckDB で実際に集計する体験ができる |
| フロント | Vite + TypeScript (バニラ) | 認知負荷を下げる |
| 可視化 | Chart.js | 正規分布カーブ + 自分の位置 |
| データ形式 | CSV (静的アセット) | 生成が依存ゼロ・DuckDB-WASM が直接読める |

## 計算式

- 偏差値 = 50 + 10 × (x − μ) / σ
- z スコア = (x − μ) / σ
- パーセンタイル = 正規分布 CDF(z) × 100（上位%はその補数）
- ※ μ, σ は疑似サンプルから DuckDB の `avg()` / `stddev_samp()` で算出する

## データ設計 (data/definitions.json)

各カテゴリ: id / label / unit / group(性別など) / mean / sd / higherIsBetter / n / source

初期カテゴリ（公表値ベースの概算・出典注記付き）:
- 身長(成人 男/女)
- 体重(成人 男/女)
- IQ(全体)
- 握力(成人 男/女, 新体力テスト)
- 50m走(成人 男/女, 秒・小さいほど良い)
